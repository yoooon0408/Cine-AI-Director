## 전체 파이프라인 흐름

```
UE5 플레이
    ↓
ExperienceLogger → Saved/CineAILogs/*.jsonl  (자동)
    ↓
ML/src/data_pipeline/collector.py            (로그 파싱)
    ↓
ML/src/data_pipeline/preprocessor.py         (전처리)
    ↓
ML/src/training/trainer.py                   (PPO 훈련)
    ↓
ML/models/exports/                           (모델 저장)
```
