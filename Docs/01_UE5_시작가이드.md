
## 준비물 체크리스트

- [ ] Unreal Engine 5.3 이상 설치 (Epic Games Launcher에서)
- [ ] Visual Studio 2022 Community 설치 (무료)
- [ ] Git 설치
- [ ] 저장공간 최소 50GB 여유

---

## Step 1 — 언리얼 프로젝트 만들기

1. **Epic Games Launcher** 실행
2. **Unreal Engine** 탭 → **Launch** 클릭
3. 새 프로젝트 화면에서:
   - `Games` 카테고리 선택
   - **`Third Person`** 템플릿 선택 ← 이게 중요! (캐릭터가 이미 포함되어 있어요)
   - C++ 선택 (Blueprint 아닌 것)
   - 프로젝트 이름: `CineAIDemo`
   - 저장 위치 지정
4. **Create** 클릭 → 프로젝트 생성 대기 (수 분 소요)

> **왜 Third Person?**
> 이미 걸어다니는 캐릭터와 기본 레벨이 포함되어 있어서,
> 우리 카메라 시스템을 바로 테스트할 수 있어요.

---

## Step 2 — 플러그인 연결하기

1. 방금 만든 `CineAIDemo` 프로젝트 폴더로 이동
2. `Plugins` 폴더 생성 (없으면)
3. `CineAIDirector` 폴더 전체를 `Plugins/` 안에 복사

```
CineAIDemo/
├── CineAIDemo.uproject
├── Content/
├── Source/
└── Plugins/
    └── CineAIDirector/       ← 여기에 복사!
        ├── CineAIDirector.uplugin
        ├── Source/
        └── ...
```

4. `CineAIDemo.uproject` 파일을 더블클릭으로 실행
5. "플러그인을 새로 발견했습니다. 다시 빌드하겠습니까?" → **Yes** 클릭
6. 빌드 완료 후 에디터가 열리면 성공!

## Step 3 — 무료 에셋 다운로드하기

에픽게임즈의 **Fab** (구 마켓플레이스)에서 무료 에셋을 받을 수 있어요.
아래 에셋들은 모두 무료이고 테스트에 딱 적합합니다.

### 방법 1 — Epic Games Launcher에서 받기

1. Epic Games Launcher → 상단 **Fab** 탭 클릭
2. 검색창에 에셋 이름 입력
3. **Free** 필터 체크
4. 에셋 페이지에서 **Add to Project** 클릭
5. `CineAIDemo` 프로젝트 선택 → **Add to Project**

### 방법 2 — 브라우저에서 fab.com 접속

1. `fab.com` 접속 후 로그인 (Epic 계정)
2. 에셋 찾기 → **Free** 필터
3. **Add to My Library** 클릭
4. Epic Games Launcher로 돌아와서 **Library → Vault**에서 다운로드

## Step 4 — 에셋을 레벨에 적용하기

### 4-1. 다운받은 에셋 확인하기

에디터 하단 **Content Browser** (콘텐츠 브라우저)에서
`Content/` 폴더 안에 다운받은 에셋이 들어있어요.

```
Content/
├── ThirdPerson/         ← 기본 캐릭터 관련
├── StarterContent/      ← 기본 큐브/재질
└── [다운받은 에셋 폴더]  ← 여기에 추가됨
```

### 4-2. 새 레벨 만들기

1. 상단 메뉴 **File → New Level**
2. **Basic** 선택 (빈 레벨 + 기본 조명)
3. 저장: `File → Save Current Level As` → `TestMap` 이름으로 저장

### 4-3. 환경 배치하기

**방법 A: 에셋 팩에 이미 완성된 맵이 있는 경우**
1. Content Browser에서 에셋 폴더 안 `Maps/` 폴더 찾기
2. `.umap` 파일 더블클릭 → 바로 열림!

**방법 B: 개별 메시 배치**
1. Content Browser에서 원하는 메시(폴리곤 모델) 찾기
2. 뷰포트(3D 화면)에 드래그 앤 드롭
3. 위치/회전/크기 조절: `W`(이동), `E`(회전), `R`(크기)

### 4-4. 카닥 바닥(Floor) 기본 설정

테스트에 필요한 최소 구성이에요:

```
반드시 있어야 할 것:
  ✅ 평평한 바닥 (캐릭터가 걸을 수 있게)
  ✅ 기본 조명 (Directional Light)
  ✅ Sky Atmosphere (하늘)

있으면 좋은 것:
  ✅ 벽/건물 (카메라 구도를 테스트하기 좋음)
  ✅ 여러 경로 (캐릭터 이동 테스트)
```

## Step 5 — AI 카메라 시스템 설치하기

이제 우리 플러그인을 레벨에 넣을 차례예요!

### 5-1. Cine AI Camera Actor 배치

1. 상단 검색창(또는 Place Actors 패널)에 `CineAI` 입력
2. **CineAI Camera** 액터가 나타나면 뷰포트에 드래그
3. 위치는 일단 레벨 중앙 어딘가에 배치

### 5-2. AI Director Actor 배치

1. 마찬가지로 **Cine AI Director** 검색 후 배치
2. 배치한 Director 액터 클릭 → 우측 **Details** 패널 열기
3. 아래 항목 설정:

```
[Details 패널 설정]

CineAI > Setup:
  Target Camera: [앞서 배치한 CineAI Camera 선택]  ← 드롭다운에서 선택
  Enable AI API:  체크 해제 (일단 테스트할 때는 꺼도 됨)

CineAI > API:
  API Key: (ML 담당자에게 받아서 입력)
```

### 5-3. 플레이어 캐릭터 확인

Third Person 템플릿에 이미 캐릭터가 있어요.
`Content/ThirdPerson/Blueprints/BP_ThirdPersonCharacter` 가 기본 캐릭터입니다.

레벨에 **Player Start** 액터가 있는지 확인하세요.
(없으면 Place Actors → `Player Start` 검색해서 배치)

## Step 6 — 첫 번째 테스트!

1. 상단 **Play** 버튼 클릭 (▶)
2. WASD로 캐릭터 이동
3. AI Director가 자동으로 카메라 각도를 바꾸는지 확인

### 테스트 확인 방법

**화면 왼쪽 상단에 로그가 보여야 해요:**
```
[CineAIDirector] 플러그인 시작
[AIDirector] API 키가 설정되지 않았습니다. 폴백 모드로 동작합니다.
[ExperienceLogger] 로그 파일: .../Saved/CineAILogs/session_xxx.jsonl
```

에러가 없으면 성공입니다!

### 아직 카메라가 안 움직여요?

- Director의 `Target Camera`가 연결됐는지 확인
- 뷰포트가 "Cine AI Camera" 시점으로 전환됐는지 확인
  - 방법: `Window → Viewports → Pilot Camera`

---

## Step 7 — 더 나은 테스트를 위한 설정

### 두 번째 캐릭터 추가하기 (대화 씬 테스트용)

1. Content Browser에서 `BP_ThirdPersonCharacter` 찾기
2. 뷰포트에 드래그 (두 번째 캐릭터)
3. 이 캐릭터는 플레이어가 아니므로 `Auto Possess Player → Disabled` 설정

### 씬 타입을 강제로 바꿔보기

`SceneAnalyzer` 컴포넌트의 `ClassifyScene` 함수에서
현재 씬 타입을 임의로 변경해서 카메라 반응을 볼 수 있어요.
(이건 ML 담당자와 같이 봐요!)

## 자주 묻는 질문

**Q: 빌드가 실패해요**
> Visual Studio 2022가 설치되어 있는지 확인하세요.
> 에러 메시지를 복사해서 ML 담당자에게 공유해주세요.

**Q: 에셋을 넣었는데 화면이 검은색이에요**
> Directional Light와 Sky Atmosphere가 레벨에 있는지 확인하세요.
> 없으면 Place Actors에서 검색해서 추가하세요.

**Q: 카메라가 이상한 곳을 보고 있어요**
> AIDirector의 폴백 로직이 아직 카메라 위치를 계산하지 않아요.
> ML 담당자에게 알려주세요.

**Q: Content Browser에 다운받은 에셋이 안 보여요**
> Epic Games Launcher → Library → Vault에서 다운로드가 완료됐는지 확인하세요.
> 완료됐다면 에디터에서 `Edit → Refresh...` 해보세요.
